
# SubPix

Uma breve descrição sobre o que esse projeto faz e para quem ele é


## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/velsreis/SubPix.git
```

Entre no diretório do projeto

```bash
  cd my-project
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run start
```
